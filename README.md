# DiamanteDeX

![](https://i.imgur.com/e0HaWS0.gif)

## Introdução
simples gerador de diamantes formados por "x" do teclado, programa que se ultiliza de 
conceitos basicos como loops,condições,validação e vários outros

## Funcionalidades

- [x] **Gerar diamante:** Gera diamantes de diferntes tamanhos baseado no **número ímpar escolhido**
 
- [x]  **Validação:** Valida se o número é **ímpar**

- [x] **Validação número inteiro:** Valida se o número é **inteiro**

## Como usar


1. **Clone o repositório:**
```
https://github.com/gustavoduara/DiamanteDeX
```

2. **Abra o terminal ou o prompt de comando e naegue até a pasta raiz**

```
cd DiamanteDeX
```

3. **Ultize o comando abaixo para rodar o programa**
```
dotnet restore
```

4. **Compile o comando**

```
dotnet build --configuration Release
```

5. **Para executar o projeto compilando em tempo real**

```
dotnet run DiamanteDeX.ConsoleApp
```

6. **Para executar o projeto compilado navegue até a pasta**
```
DiamanteDeX.ConsoleApp.exe
```

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
